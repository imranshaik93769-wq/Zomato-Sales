### End-to-End Data Analytics Project using Excel • MySQL • Power BI

# Project Overview

This project demonstrates an **end-to-end Data Analytics workflow** using real-world business scenarios from a Zomato sales dataset.

The project begins with raw CSV files, followed by data cleaning and validation using **Microsoft Excel**, relational database design and business analysis using **MySQL**, and finally an interactive dashboard developed in **Microsoft Power BI**.

The dashboard helps analyze sales performance, customer spending, restaurant operations, delivery status, monthly order trends, and revenue using interactive visualizations.

---

# Objectives

- Import and clean raw sales data
- Build a relational database using MySQL
- Establish Primary Key and Foreign Key relationships
- Perform SQL-based business analysis
- Create an interactive Power BI dashboard
- Generate meaningful business insights
- Improve decision-making through data visualization

# SQL Analysis

The following business problems were solved using MySQL.

### Top 3 Most Ordered Food Items

Used CTE and Window Functions to identify the top ordered menu items.

---

### Most Popular Food Item in Top Restaurant

Identified the highest selling food item from the best-performing restaurant.

---

### Rider Performance Analysis

Found the rider who delivered the highest number of Veg Biryani orders.

---

### Customer Restaurant Analysis

Calculated the total number of orders placed by a specific customer in a selected restaurant.

---

### Peak Order Time Analysis

Identified peak ordering hours using 2-hour time intervals.

---

### Average Order Value (AOV)

Calculated average order value for high-frequency customers.

---

### High Value Customers

Identified customers who spent more than ₹10,000.

---

### Monthly Top Customers

Ranked the top 3 highest spending customers for every month.

---

### Delivery Performance

Analyzed restaurants with the highest number of "Not Delivered" orders.

---

### Customer Lifetime Value (CLV)

Calculated total revenue generated by every customer.
